# Privacy Policy

**Last Updated: January 30, 2026**

## RetroVision Cabs Converter

This Privacy Policy describes how RetroVision Cabs Converter ("the App") handles your information.

## Information We Do NOT Collect

RetroVision Cabs Converter is designed with privacy in mind:

- **No Personal Data Collection**: The App does not collect, store, or transmit any personal information.
- **No Analytics**: The App does not use analytics services or tracking technologies.
- **No Account Required**: The App does not require account creation or login.
- **No Network Transmission**: Your data stays on your device and is never sent to external servers.
- **No Advertising**: The App contains no advertisements and does not use advertising tracking.

## Information That Stays on Your Device

The App processes the following data locally on your device:

### File Access
- **Cabinet Asset Files**: 3D models (GLB, USDZ), images (PNG, JPG), videos (MP4, MKV), and configuration files (YAML, JSON) that you choose to import.
- **Output Files**: Converted USDZ files, Swift code, and documentation generated by the App.
- **Temporary Files**: Cached previews, extraction folders, and processing data stored in temporary directories.

### Application Data
- **User Preferences**: Settings such as folder paths and Blender location are stored in macOS User Defaults on your device only.
- **Security Logs**: Local logs for debugging purposes, stored in your workspace folder.

## Third-Party Tools

The App uses the following external tools that you must install separately:

- **Blender**: An open-source 3D creation suite (https://www.blender.org)
- **FFmpeg**: An open-source multimedia framework (https://ffmpeg.org)
- **Python**: The Python programming language (https://www.python.org)

These tools are executed locally on your device. We recommend reviewing the privacy policies of these third-party tools.

## File System Access

The App requires access to:
- **User-Selected Folders**: To read source cabinet files and write converted outputs
- **External Volumes**: To access files on external drives
- **Temporary Directories**: For processing and caching

All file access is user-initiated and limited to folders you explicitly select.

## Data Security

- All file processing occurs locally on your device
- No data is transmitted over the internet
- Temporary files are created with appropriate permissions
- ZIP files are validated before extraction to prevent security issues

## Children's Privacy

The App does not knowingly collect any personal information from children under 13 years of age. The App is a development tool intended for use by adults.

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date above.
